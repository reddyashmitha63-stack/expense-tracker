# Expense Tracker

A responsive Expense Tracker application built using HTML, CSS, and JavaScript.

## Features

- Add expenses
- Delete expenses
- Expense categories
- Total spending calculation
- Budget tracking
- Remaining budget display
- Budget usage warnings
- Budget exceeded alerts
- Category-wise analytics
- Highest spending category
- Average expense calculation
- Local Storage support

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage API
